# Grain Marketplace Frontend

This is the frontend application for the Grain Marketplace platform.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Environment Variables

The frontend connects to the backend API through Vite's proxy configuration.
Backend URL is configured in `vite.config.js`.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
